# Kade-Engine-Charts-to-Psych-Engine-Converter
A converter that is capable of converting Kade Engine Charts! You need to edit the stage though. VERY useful for porting mods.

# What this can do
It can port pretty much any chart there is. I might create a Psych Engine to Kade.

I **advise you** to go test it for yourself!
